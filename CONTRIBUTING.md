# Contributing to the Cruz Creative Marketing Hub

## Naming branches

| Type | Pattern | Example |
|---|---|---|
| New asset or campaign | `add/` | `add/q3-social-templates` |
| Update to existing content | `update/` | `update/brand-color-palette` |
| Fix a mistake | `fix/` | `fix/homepage-copy-typo` |

## Commit messages

Use plain English. Start with a verb. Keep it under 72 characters.

```
Add Instagram story templates for July campaign
Update brand guide with new secondary typeface
Fix broken link in landing page footer
```

## Pull request checklist

Before opening a PR, confirm:

- [ ] Files are in the right folder (`brand/`, `social/`, etc.)
- [ ] No sensitive info (passwords, client PII) in any file
- [ ] PR description explains *what* changed and *why*
