# Cruz Creative — Marketing Hub

Central repository for all marketing activities at Cruz Creative Agency.

## What's in here

| Folder | Purpose |
|---|---|
| `brand/` | Brand guides, logos, color palettes, typography |
| `landing-pages/` | Landing page code and copy |
| `social/` | Social media post copy, templates, and schedules |
| `tools/` | Marketing automation scripts and utilities |

## Getting started

Clone the repo and you have everything in one place:

```bash
git clone https://github.com/cruz-creative-marketing/marketing-hub.git
cd marketing-hub
```

## Contributing

1. Create a branch: `git checkout -b your-feature`
2. Make your changes
3. Open a pull request — every change goes through review before hitting `main`

See [CONTRIBUTING.md](CONTRIBUTING.md) for branch naming and commit message conventions.
